Syrus Google Custom Search

== Description ==

This plugin works with GCS for elaborate a maximum of 5 research results to append to wordpress searchform
Tested in Firefox, Safari, Chrome and IE.

== Installation ==

The only thing you need to do: copy Syrus-Google-Custom-Search.php into your /wp-content/plugins directory.
You will see related search results, and so, GCS results powered by GoogleSearchEngine, under your each
Wordpress search form when you'll insert research terms
